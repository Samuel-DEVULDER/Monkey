# Monkey

3D Mokey skull rotating with 3 light sources for the Amiga. 
100% software. Works on mc680x0+fpu including ac68080.

Inspired from: http://github.com/halcy/rasterizer

Supports many (Amiga) compilers:
* Makefile.295
	* gcc-2.95.1
* Makefile.322
	* gcc-3.2.2
* Makefile.346
	* gcc-3.4.6
* Makefile.6.3.1
	* gcc-6.3.1
* Makefile.641
	* gcc-6.4.1
* Makefile.vbcc
	* vbcc
* Makefile.sasc
	*  SAS/C
